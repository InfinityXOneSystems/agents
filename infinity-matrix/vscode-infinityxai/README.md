# InfinityXAI VS Code Extension

This extension adds a deep blue electric triangle icon to the VS Code activity bar. Clicking the icon or the "Open InfinityXAI Admin" command will launch the InfinityXAI admin dashboard at [https://infinityxai.com/admin](https://infinityxai.com/admin).

## Features
- Custom activity bar icon (deep blue electric triangle)
- Activity bar view with a button to open the admin dashboard
- Command palette command: "Open InfinityXAI Admin"

## Usage
1. Install this extension in VS Code.
2. Click the electric triangle icon in the activity bar (left sidebar).
3. Use the button in the view or the command palette to open the dashboard.

## Icon
The icon is located at `media/electric-triangle.svg`.
