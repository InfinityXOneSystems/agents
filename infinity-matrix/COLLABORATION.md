# Collaboration Guide

## Agent Roles and Responsibilities

### 1. User (Owner)
- Oversees the entire project.
- Provides high-level direction and priorities.
- Reviews and approves changes.

### 2. VS Code Copilot (Local/DevOps Agent)
- Assists with local development tasks.
- Automates repetitive coding and debugging processes.
- Manages local environment setup and validation.

### 3. GitHub Copilot (Remote)
- Acts as the system architect and orchestrator.
- Manages GitHub Actions workflows and CI/CD pipelines.
- Writes, refactors, and optimizes code for the project.

---

## Communication and Handoff
- Use clear, atomic pull requests for changes.
- Document all workflows and processes in `/docs`.
- Ensure seamless integration between local and remote agents.

---

## Daily Operations
- Sync local changes with GitHub regularly.
- Monitor CI/CD pipelines for issues.
- Update documentation as the system evolves.
