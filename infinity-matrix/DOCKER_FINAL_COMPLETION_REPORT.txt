## üîë Key Commands Reference

```bash
# Start/Stop
docker-compose up -d              # Start (background)
docker-compose stop               # Stop gracefully
docker-compose down               # Stop & remove

# Monitor
docker-compose ps                 # Service status
docker-compose logs -f            # View logs
docker stats                       # Resource usage
python docker_health_check.py     # Health check

# Develop
docker-compose build              # Rebuild images
docker-compose build orchestration # Rebuild one service
# Hot-reload:
docker-compose -f docker-compose.yml -f docker-compose.dev.yml up

# Maintain
docker-compose restart            # Restart all
docker-compose exec frontend sh   # Access container shell
docker-compose down -v            # Full cleanup
```
