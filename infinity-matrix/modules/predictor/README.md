# InfinityXAI Predictor Submodule

This module provides a fully autonomous financial prediction system with:
- High-accuracy prediction agent (Vision Cortex + Vertex AI)
- Advanced paper trading (stocks, crypto, Infinity Coin testnet)
- Auto/hybrid/manual trading modes
- Admin access restricted to /admin
- Day trading specialist agent for daily predictions

## Features
- Predictive analytics dashboard at `/predict`
- Paper trading simulator with performance tracking
- Crypto trading for Infinity Coin (testnet)
- Autonomous agent for daily trading
- Branded UI matching infinityxai.com

## Setup
- Integrate with main app via import or submodule
- Configure API keys for Vertex AI, Vision Cortex
- Deploy with Docker/CI/CD

## Usage
- End users access `/predict` for dashboard and trading
- Admins access `/admin` for management
