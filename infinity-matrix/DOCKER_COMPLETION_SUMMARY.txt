## üí° Pro Tips

### For Development
```bash
# Use hot-reload (auto-reload on code changes)
docker-compose -f docker-compose.yml -f docker-compose.dev.yml up

# Changes appear automatically within 2 seconds
# Edit files in frontend_stack/frontend/src/ and orchestration/
```

### For Performance Monitoring
```bash
# Watch resource usage in real-time
docker stats --no-stream=false

# View logs with timestamps
docker-compose logs -f --timestamps

# Check recent errors
docker-compose logs | grep ERROR
```

### For Troubleshooting
```bash
# Full logs with context
docker-compose logs -f | grep -A 5 -B 5 "ERROR"

# Specific service logs
docker-compose logs orchestration | head -50

# Access container shell (for debugging)
docker-compose exec frontend sh
docker-compose exec orchestration bash
```

### For Updates
```bash
# Rebuild after code changes
docker-compose build --no-cache
docker-compose up -d

# Or in one command
docker-compose build --no-cache && docker-compose up -d
```
